# Micro Nutrient Deficiency Detection

This project detects micronutrient deficiencies in banana plants using image processing and machine learning.

## Features
- Upload banana leaf image
- Detect nutrient deficiency
- Web-based interface
- Flask backend with ML model

## Technologies Used
- Python
- Flask
- TensorFlow / Keras
- OpenCV
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository
2. Create virtual environment
3. Install dependencies using `pip install -r requirements.txt`
4. Run `python app.py`
5. Open `index.html` using Live Server

## Note
The trained model file is not included due to size limitations.
